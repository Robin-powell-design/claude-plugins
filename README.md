# Robin Powell Design Plugins

A collection of premium UI design plugins for Claude Code.

## Plugins

### premium-ui-designer
Design and build premium UI with the craft level of Linear, Apple, Stripe, and Vercel. Supports both dark and light themes. Produces single-file HTML with zero external images.

### social-media
Create social media visuals as single-file HTML - posts, stories, carousels, banners, and cards for Instagram, LinkedIn, X/Twitter, and other platforms.

### refactoring-ui
Expert UI design guidance combining principles from Refactoring UI (Adam Wathan & Steve Schoger), Practical UI (Adham Dannaway), Picture This (Molly Bang), and Grid Systems in Graphic Design (Josef Muller-Brockmann).

## Installation

Add this marketplace to Claude Code:

```bash
claude plugin marketplace add Robin-powell-design/claude-plugins
```

Then install individual plugins:

```bash
claude plugin install premium-ui-designer
claude plugin install social-media
claude plugin install refactoring-ui
```
